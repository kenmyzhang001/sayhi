# 短信模板生成系统

完整的短信模板生成系统，支持多种生成方式和字符编码。

## 项目结构

```
sayhi/
├── backend/          # Go 后端服务
├── frontend/         # Vue 前端应用
└── 需求文档-短信模板生成系统.md
```

## 快速开始

### 后端启动

```bash
cd backend
go mod download
go run main.go
```

后端服务运行在 `http://localhost:8080`

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

前端应用运行在 `http://localhost:3000`

## 功能特性

### 核心功能
- ✅ 模板解析（支持括号位置和范围值）
- ✅ 顺序生成和随机生成
- ✅ 多种字符编码支持（ASCII、Zawgyi、Unicode、其它）
- ✅ 字符数统计和超出提示
- ✅ 内容编辑功能
- ✅ 位置值后台配置管理

### 技术特点
- **后端**：Go + Gin，RESTful API，内存存储（可扩展为数据库）
- **前端**：Vue 3 + Element Plus，响应式设计，实时反馈

## 使用示例

### 模板格式
```
(1)(baidu.com)(2)(3-10)
```

### 生成结果
- **顺序生成**：按 a → b → c → d 顺序生成所有组合
- **随机生成**：随机打乱位置顺序生成

### 字符限制
- 每条短信限制 70 字符
- 超出部分会明确标识
- 支持编辑超出内容
